
📷 智能图像标注与校正工具

一款专业、易用的图像处理桌面应用，专为需要精确标注、透视校正和图像增强的用户打造。

核心功能：

    灵活标注：点击图像任意位置即可添加红色标记点，右侧列表框实时显示所有点坐标，支持批量删除与一键清空。
    透视校正：选定图像中四个角点，一键完成仿射变换，将倾斜或扭曲的平面矫正为矩形，输出校正后图片。
    图像增强：内置清晰度增强算法（如去模糊、对比度提升），轻松改善图像质量。
    双语界面：中英文一键切换，满足国内外用户需求。
    智能预览：校正/增强后弹出新窗口预览，提供“保存图片”、“更新主图”、“保存并更新”三种灵活选项。

适用场景：
    文档扫描件校正（合同、表单、书籍）
    计算机视觉数据标注（目标检测、车牌识别）
    摄影后期处理（透视调整、细节增强）
    学术科研（实验图像标注、几何校正）

软件特点：

    🔒 完全离线运行，不上传任何用户数据 – 所有操作均在本地完成，无需联网，保护隐私安全。
    🚀 轻量级、无需安装额外依赖（已打包为独立 .exe）
    🎯 鼠标点击即标注，坐标精确到像素
    🎨 现代深色主题界面，按钮彩色区分，操作反馈清晰
    💾 支持 PNG、JPEG、BMP 等常见图像格式

📷 Smart Image Annotation & Correction Tool

A professional and easy-to-use desktop application for precise image annotation, perspective correction, and clarity enhancement.

Key Features:

Flexible Annotation: Click anywhere on the image to add red marker points; real-time coordinate list with batch deletion and one‑click clear.
Perspective Correction: Select 4 corners, perform affine transformation to rectify tilted or skewed planes into perfect rectangles.
Image Enhancement: Built‑in clarity enhancement (deblurring, contrast boost) to improve image quality.

Bilingual UI: Instant switch between Chinese and English.

Smart Preview: Pop‑up window after correction/enhancement with three saving options (Save, Update Main, Save & Update).

Use Cases:

    Document scanning (contracts, forms, books)
    Computer vision data annotation (object detection, license plate recognition)
    Photo post‑processing (perspective adjustment, detail enhancement)
    Academic research (experimental image annotation, geometric correction)

Highlights:

    🔒 Fully offline – no user data uploaded – All processing is done locally; no internet connection required, ensuring your privacy.
    🚀 Lightweight, standalone .exe – no extra dependencies
    🎯 Click‑to‑annotate, pixel‑accurate coordinates
    🎨 Modern dark theme, color‑coded buttons, clear feedback
    💾 Supports PNG, JPEG, BMP, and more
